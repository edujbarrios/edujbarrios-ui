# edujbarrios-ui

edujbarrios-ui is a collection of self-made UI / frontend components ready to copy, paste, and use. Built with Next.js, TypeScript, and Tailwind CSS by Eduardo J. Barrios.

## Overview

edujbarrios-ui is a dark-mode-first open-source component gallery for modern frontend developers, AI product builders, and creative engineers. It showcases handcrafted React components with live previews, source code, usage examples, and practical accessibility notes.

## Features

- Polished landing page with a premium dark interface
- Searchable and filterable component gallery
- Individual component pages with live previews and source code
- Copy-to-clipboard interactions with visual feedback
- Local TypeScript registry for component metadata
- Responsive navigation and mobile-friendly layouts
- No backend and no database
- Ready for GitHub and Vercel

## Tech Stack

- Next.js App Router
- TypeScript
- React
- Tailwind CSS
- Local TypeScript data

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the project locally.

## Development

```bash
npm run dev
npm run lint
npm run build
```

Component metadata lives in `lib/components.ts`, source snippets live in `data/component-code.ts`, and preview components live in `components/ui`.

## Deployment

The source code is intended to live publicly on GitHub at:

```text
https://github.com/edujbarrios/edujbarrios-ui
```

The rendered version is intended to be deployed on Vercel at:

```text
https://edujbarrios-ui.vercel.app
```
